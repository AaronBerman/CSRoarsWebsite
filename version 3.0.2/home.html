Problem Statement

Collection, storage, and processing of individualized data, in regards to electronic reference material, is fragmented and/or domain specific (such as finance and enterprise). Educators and researchers need a framework and tool-set to both collect and process data for the purpose of knowledge management.

Solution Statement

Our goal is to build an extensible framework (ROARS) and a tool-set with which to collect and store electronic data in an organized structure, regardless of format or type. The framework can be extended throught he use of api calls. The base tool-set would include a webscraping utility, a utility to compare two versions of the same document, and a way to track changes to the document. The framework will allow for modules to be built for other types of processing of data after the initial storage and collection .
